class eh utilizado para ser acessado no css para dar estilos ao elemento ou ser acessado pelo javascript para conferir logicas.
Permite que o conteudo seja editado pelo usuario.
data-* tambem eh utilizado para ser acessado pelo css e javascript como as classes e IDs.
O hidden esconde um elemento.
IDs sao dentificadores unicos de elementos. Nao podem ser repetidos por pagina.
O atributo style serve para estilizar prioritariamente um elemento HTML (pouco utilizado).
O tabindex eh uma importante ferramenta para definir a ordem que elementos sao acessados pelo usuario quando este apertar o botao tab do teclado.
Terceiro tab
Primeiro tab
Segundo tab
O titulo aparece quando o usario deixa o cursor do mouse parado em cima do elemento.
